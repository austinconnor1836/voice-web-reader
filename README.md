# Lexi - Voice Web Reader

A powerful Chrome extension that reads web pages aloud with sentence-level highlighting. Lexi uses the native Web Speech API to provide a smooth text-to-speech experience completely free of charge.

## Features

-   **Smart Highlighting:** Highlights the exact sentence being read.
-   **Click to Start:** Highlight any text on the page to start reading from that point.
-   **Instant Controls:** Change voice and speed on the fly without restarting.
-   **Native Voices:** Uses your browser's built-in voices (offline compatible).
-   **Dynamic Content:** Handles complex page layouts and updates.

## Installation

1.  Clone this repository.
2.  Open Chrome and navigate to `chrome://extensions`.
3.  Enable **Developer mode** (top right).
4.  Click **Load unpacked**.
5.  Select the `lexi-reader` folder from this repository.

## Usage

1.  Click the Lexi icon in the toolbar.
2.  Press **Play** to start reading from the beginning.
3.  **Or** select any text on the page to start reading from that selection.
4.  Adjust **Speed** and **Voice** in the popup menu.
